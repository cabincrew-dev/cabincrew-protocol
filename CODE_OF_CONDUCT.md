# CabinCrew Protocol – Code of Conduct
Version: draft

This Code of Conduct applies to all contributors, maintainers, users, and participants in the CabinCrew Protocol community and project spaces, including GitHub repositories, issues, pull requests, discussions, and related communication channels.

The goal of this Code of Conduct is to ensure a welcoming, safe, and respectful environment for everyone.

---

## 1. Our Standards

### We commit to maintaining an environment where people:
- Act with respect, kindness, and professionalism.
- Provide constructive feedback and accept feedback gracefully.
- Use welcoming and inclusive language.
- Help create a collaborative, supportive community.

### We do **not** tolerate:
- Harassment, discrimination, or abusive behavior.
- Personal attacks or insults.
- Demeaning or derogatory comments.
- Threats or encouragement of harm.
- Disruption of discussions or project activities.
- Publishing others’ private information without consent.

---

## 2. Scope

This Code of Conduct applies to:
- All GitHub interactions (issues, PRs, discussions).
- All communication related to the project.
- All contributions, whether public or private.
- Any space where the CabinCrew Protocol project is represented.

---

## 3. Responsibilities of Maintainers

Project maintainers are responsible for:
- Enforcing the Code of Conduct.
- Reviewing violations in a fair and impartial manner.
- Taking appropriate corrective action, which may include:
  - Warning the offender
  - Removing comments
  - Rejecting contributions
  - Temporary or permanent banning from participation

Maintainers may remove or edit comments that violate this Code.

---

## 4. Reporting Violations

If you observe or experience a violation:
- Please open a private GitHub contact request to maintainers.

Reports should include:
- A description of the incident  
- Links, screenshots, or other evidence  
- Your contact information (optional but helpful)

All reports will be handled confidentially.

---

## 5. Enforcement Guidelines

Violations may result in one or more of the following actions:

### **5.1 Correction**
A private or public notice explaining why behavior was inappropriate.

### **5.2 Warning**
A firm warning of consequences if behavior continues.

### **5.3 Temporary Ban**
A period during which the contributor cannot interact with the project.

### **5.4 Permanent Ban**
Complete removal from project participation.

The severity of the response depends on:
- Impact of behavior  
- Recurrence  
- Intent  
- Safety concerns  

---

## 6. No Retaliation

The project prohibits retaliation against anyone who:
- Reports a violation in good faith  
- Assists in an investigation  
- Helps enforce this Code  

Retaliation will itself be treated as a violation.

---

---

## 8. Acceptance

By participating in this project, you agree to follow this Code of Conduct.

Thank you for helping make the CabinCrew Protocol a respectful, collaborative, and safe community.
